Create or update recipe

Creates or update a recipe

Special Cases:

  • To create a new recipe from scratch:
    • Provide ingredientQuantities (will be normalized if normalizeQuantity=true), you can optionally provide a title
  • To only change title:
    • Provide parent WITHOUT ingredientQuantities and with title
  • To update a recipe:
  • Provide a parent and ingredientQuantities
Authorizations:
ApiKeyAuth
Request Body schema: application/json
required
title
string non-empty

The name/title of the new recipe (required for all requests)

ingredientQuantities
Array of numbers <float> [ items <float > >= 0 ]

Array of quantity values (required when creating new recipe). Will be normalized to sum to 1.0 if normalizeQuantity=true.

parent
string

ID of existing recipe to copy from (alternative to quantities). When provided, only the title can be modified.

normalizeQuantity
boolean
Default: true

Automatically normalizes quantities to sum to 1.0 (default: true). Ignored when using parent recipe.

Responses

Production Server

https://symphony.fr/api/v2/recipes

Local Development Server (For internal testing)

https://localhost:3000/api/v2/recipes

Discover new recipes

Allows you to find new recipes based on various criteria using cursor-based pagination. Provide both lastCursorId and lastCursorScore from the previous page's pagination info to fetch the next set of results.

Authorizations:
ApiKeyAuth
query Parameters
category
string
Default: "all"
Example: category=vegetarian

Filter recipes by category.

lastCursorId
string
Example: lastCursorId=63485116b0ff720c5cb04bea

The ID of the last recipe retrieved from the previous page (obtained from 'pagination.lastCursorId'). Omit for the first page.

lastCursorScore
number <float>
Example: lastCursorScore=0.85

The score of the last recipe retrieved from the previous page (obtained from 'pagination.lastCursorScore'). Omit for the first page.

timescale
integer >= 1
Default: 30
Example: timescale=7

Time window in days for considering recipe relevance or recency. Must be a positive integer.

limit
integer [ 1 .. 100 ]
Default: 20
Example: limit=20

Maximum number of recipes to return per page.

lang
string
Default: "fr"
Enum: "en" "fr"

The language for localized response fields (e.g., ingredient titles, recipe titles)

Responses

Production Server

https://symphony.fr/api/v2/recipes/discover

Local Development Server (For internal testing)

https://localhost:3000/api/v2/recipes/discover

Add one or more meals to the upcoming basket

Adds one or more meals based on recipe IDs (optionally with specific quantities) to the upcoming basket. Returns the details of the meals, including any non-critical warnings (e.g., low popularity, inclusion of CBD). Critical issues (Out of stock ingredients) and invalid recipes will result in an error response.

Authorizations:
ApiKeyAuth
path Parameters
userId
required
string
Example: @me

The ID of the user. Can be a user ID or the literal string '@me' to refer to the currently authenticated user making the request.

Request Body schema: application/json
required

An array of meals to add to the basket.

Array (non-empty)
recipeId
required
string

The unique identifier of the recipe to base the meal on.

quantity
integer <int32> [ 280 .. 650 ]

Optional quantity of the meal in grams. If omitted, it is calculated based on your caloric target.

Responses

Production Server

https://symphony.fr/api/v2/users/{userId}/meals/upcoming-basket

Remove meals from upcoming basket

Removes one or more meals from a user's upcoming basket. Provide an array of meal IDs in the request body. Returns a 400 error if any of the provided meal IDs are invalid or not found in the basket.

Authorizations:
ApiKeyAuth
path Parameters
userId
required
string
Example: @me

The ID of the user. Can be a user ID or the literal string '@me' to refer to the currently authenticated user making the request.

Request Body schema: application/json
required

An array containing the unique IDs of the meals to be removed from the upcoming basket.

Array (non-empty)
string

The unique identifier of a meal instance within the basket.
